User guide
==========

.. _installation:

Installation
------------

To use SuperDSM, first install it using conda:

.. code-block:: console

   conda install -c bioconda superdsm

Usage
-----

Batch processing
****************

To run SuperDSM from command line, use:

.. code-block:: console

   python -m 'gocell.batch'

To export (intermediate) results, use:

.. code-block:: console

   python -m 'gocell.export'
